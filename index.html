원하는 실험을 시작해 보세요.

이 `index.html`은 시작점일 뿐입니다. 내용을 자유롭게 바꾸고, 스타일을 수정하고, JS 파일을 추가해 보세요. 그 과정에서 GitHub Desktop으로 변경 이력을 남기며 연습할 수 있습니다.

이 파일을 수정하고 저장한 뒤, GitHub Desktop에서 변화를 확인해 보세요.

에디터에서 이 파일을 열고 실시간 미리보기를 보고 싶다면 브라우저에서 이 파일을 직접 열거나, 간단한 개발 서버를 띄워 둘 수 있습니다. (예: VS Code Live Server 등)

GET STARTED
  1. 1
    `index.html` 수정하기 텍스트를 고치거나, 색상을 바꾸거나, 새로운 섹션을 추가해 보세요.
  2. 2
    GitHub Desktop 확인 변경 사항이 어떻게 표시되는지 확인하고 첫 커밋을 만들어 보세요.
  3. 3
    브랜치 만들어 보기 새 브랜치를 만들어 다른 아이디어를 실험한 뒤, 다시 병합해 볼 수 있습니다.
터미널에서 로컬 서버 실행 예시
python -m http.server 8000
